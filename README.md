# Wallet Service Demo

A digital wallet backend service built with Python and SQL Server.

## Overview

This project demonstrates the architecture and business logic behind a digital wallet platform used for balance management, financial transactions, and payment processing.

The service provides APIs for wallet operations while ensuring transaction consistency and reliability.

## Key Features

### Wallet Management

* Create Wallet
* Wallet Balance Tracking
* Wallet Status Management

### Financial Operations

* Top-up Balance
* Withdraw Funds
* Transfer Funds
* Transaction Validation

### Transaction Management

* Transaction History
* Balance Calculation
* Payment Processing
* Financial Reconciliation

### API Services

* RESTful APIs
* Authentication & Authorization
* Business Rule Validation
* Error Handling

## Technology Stack

* Python
* SQL Server
* REST API
* RabbitMQ
* Git

## Architecture

The application follows a modular backend architecture consisting of:

* API Layer
* Business Logic Layer
* Data Access Layer
* Messaging Layer
* Transaction Layer

## Project Goals

This repository demonstrates wallet system design, transaction processing, API development, and backend service architecture concepts.
